**Python-Project**

The Cafe Management System is a simple Python-based project designed to simulate the basic operations of a cafe. The program allows users to view available cuisines, display menu items, calculate monthly expenses, and determine the profit based on revenue and expenses.

This project was created to demonstrate the use of Python fundamentals and Object-Oriented Programming (OOPS) concepts in solving a small real-world business problem.

**Project Objective:**

The objective of this project is to build a simple system that helps in:

* Organizing cafe menu items by cuisine
* Displaying available food and beverage items
* Managing monthly expenses.
* Calculating total expenses and profit
* Demonstrating Python programming concepts
**Features**

* Displays available cuisines in the cafe
* Shows menu items for the selected cuisine
* Calculates monthly expenses (inventory, salary, other expenses)
* Accepts monthly revenue input
* Calculates and displays the monthly profit
* Menu-based program structure

**Technologies Used**

* Python
* Object-Oriented Programming (OOPS)
* Lists
* Dictionaries
* Functions
* User Input Handling

**Project Structure**

* The project is organized using multiple classes:
* Menu Class – Displays available cuisines in the cafe
* Items Class – Shows menu items under each cuisine
* Expenses Class – Calculates total monthly expenses
* Revenue Class – Calculates profit based on revenue and expenses

**How the Program Works**

* The user enters the month.
* The program displays available cuisines.
* The user selects a cuisine.
* The program shows menu items for that cuisine.
* The user enters expenses and monthly revenue.
* The system calculates:
* Total Expenses
* Monthly Profit

**Learning Outcomes**

* Through this project, I practiced:
* Python programming fundamentals
* Object-Oriented Programming concepts
* Data structures like lists and dictionaries
* Structuring a real-world mini project
* Building a simple business logic program
